Detailed Description

ChessPiece – This class is a blueprint of a chess piece object. These objects can be placed on boards of type 'Board'. Each ChessPiece object contains a pos (position) and a boolean 'player1' indicating whether it belongs to player 1 or not. The 'pos' is a 1-D int[] containing the piece's position on a multi-dimensional Board.
