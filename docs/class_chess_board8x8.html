Detailed Description

ChessBoard8x8 – Implementation of a Chess Board extended from the Board Class. Contains of a 8x8 2D Board. Follows the rules of basic Chess. Implements functionality such as moving/capturing pieces, checking if a piece is occupied, initializing the board, checking if the player's king is in check, if the game is over (checkmate or stalemate), etc.
